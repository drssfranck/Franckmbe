# Portfolio Franck MBE - Data Analyst / Data Engineer

Portfolio professionnel développé avec React, Vite et TailwindCSS.

## 🚀 Technologies

- React 18
- Vite
- TailwindCSS
- Framer Motion
- i18next (FR/EN)
- Architecture modulaire

## 🎯 Fonctionnalités

- Design inspiré des dashboards data
- Mode sombre/clair
- Multilingue FR/EN
- Animations fluides
- CV interactif
- Filtrage des projets
- Formulaire de contact

## 📦 Installation

```bash
npm install
